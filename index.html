Javascript y data, vía p5.js
Introducción a la programación del lado del cliente

Lo primero que haremos en esta clase será asomarnos a la Consola de Javascript. Gracias a p5/cero.js, podrán ver en su Consola se imprimen algunos datos sobre los más acaparadores de los premios Óscar, según consulta a la data dentro un CSV que dejé en línea.

En el párrafo anterior, usé adrede las palabras data y datos; cada vez que diga data estaré refiriéndome a eso cuya forma aún está por des-velarse. Cuando diga datos estaré suponiendo que hay una forma que ya se des-vela, que permite in-formar-nos de algo concreto.

Desarrollemos la idea de diferenciar data y datos en un ejemplo de tres pasos:

Primero: Es difícil que te informes si te enfrentas solamente a data:

1934,It Happened One Night,5,5

Segundo: A condición de estar informado de que los ítems en la primera línea describen el orden de la segunda línea, el asunto es distinto: 

year, movie, oscars, nominations
1934, It Happened One Night, 5, 5

Tercero: Si cumples con la condición recién indicada, te informas del dato:

En 1934, la película It Happened One Night ganó 5 premios Óscar de los 5 a los que fue nominada.

Ahora bien, en la Consola de JavaScript no se está imprimiendo tal frase, sino:

El año 1934, la película «It Happened One Night» se llevó 5 premios Oscar para su casa.

Se imprime de tal manera, porque así es como lo indica p5/cero.js

var data;
//lo primero es crear una variable global, a la que se le asignará un contenido en preload(){}

function preload() {
  data = loadTable("https://raw.githubusercontent.com/profesorfaco/dgp602.data/gh-pages/data/acaparadores-de-premios-oscar.csv", "csv", "header");
}

function setup() {
  for (var i = 0; i < data.getRowCount(); i++) {
    var year = data.get(i, 0);
    var title = data.get(i, 1);
    var how_many = data.get(i, 2);
    print("El año " + year + ", la película «" + title + "» se llevó " + how_many + " premios Oscar para su casa.");
    print("La tabla consultada tiene " + data.getRowCount() + " filas y " + data.getColumnCount() + " columnas.");
  }
}
